# 📊 Sales Data Analysis Project

## 🚀 Overview

This project performs **data analysis on 12 months of e-commerce sales data** to uncover business insights such as sales trends, customer behavior, and product performance.

The dataset contains real-world sales records including product details, order dates, prices, and locations.

---

## 📁 Dataset

* Source: Kaggle / GitHub dataset
* Contains: 12 CSV files (January to December 2019)
* Features include:

  * Order ID
  * Product
  * Quantity Ordered
  * Price Each
  * Order Date
  * Purchase Address

---

## 🛠️ Technologies Used

* Python 🐍
* Pandas (Data Cleaning & Analysis)
* Matplotlib (Visualization)
* Seaborn (Advanced Visualization)

---

## 🔧 Project Workflow

### 1️⃣ Data Collection

* Combined 12 monthly CSV files into a single dataset

### 2️⃣ Data Cleaning

* Removed null values
* Removed invalid rows
* Converted data types

### 3️⃣ Feature Engineering

* Extracted:

  * Month
  * Hour
  * City
  * Day of week
* Created:

  * Sales column

### 4️⃣ Data Analysis (GroupBy)

* Monthly sales trends
* Sales by city
* Best-selling products
* Average product prices

### 5️⃣ Data Visualization

* 📊 Monthly Sales Bar Chart
* 📉 Scatter Plot (Quantity vs Price)
* 🌆 Sales by City
* 🛒 Best Selling Products
* 📈 Product Revenue (Horizontal Bar)
* 🔥 Heatmap (Correlation Analysis)
* ⏰ Best Advertisement Time (Area + Line Plot)

### 6️⃣ Advanced Analysis

* Identified products frequently sold together

---

## 📈 Key Insights

* 📅 Sales peak in **December** due to holiday demand
* 🌆 Major cities generate the highest revenue
* ⏰ Best time for ads: **11 AM & 7 PM**
* 💰 Lower-priced products sell in higher quantities
* 🔗 Certain products are often bought together
